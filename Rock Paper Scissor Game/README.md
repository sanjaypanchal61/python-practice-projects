# ✊✋✌️ Rock Paper Scissors Game

## 📌 Project Overview

This is a simple command-line based Rock-Paper-Scissors game built using Python.
It helps beginners understand conditional logic and user interaction.

---

## ⚙️ Features

* 🎮 User vs Computer gameplay
* 🤖 Random computer move generation
* ⚠️ Input validation (invalid choice detection)
* 🧠 Simple and clear game logic

---

## 🧠 Concepts Used

* Python `random` module
* Lists (for storing game items)
* Conditional statements (`if-elif`)
* User input handling

---

## 💻 Sample Output

```
Enter your move: Rock, Paper, Scissor = rock
User choice = rock | Computer choice = paper
Paper covers Rock , Computer win
```

---

## 🔄 Workflow Explanation

1. User inputs their choice (rock, paper, scissor)
2. Input is converted to lowercase
3. Computer randomly selects a move
4. Input is validated
5. Game logic compares both choices
6. Result is displayed (Win / Lose / Tie)
