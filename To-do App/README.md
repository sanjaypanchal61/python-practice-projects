# 📝 To-Do List App

## 📌 Project Overview

This is a simple To-Do List application built using Python.

This project is designed for beginners to understand core Python concepts like:

* Lists
* Loops
* Conditional statements
* Functions

---

## ⚙️ Features

* ➕ Add new tasks
* ✏️ Update existing tasks
* ❌ Delete tasks
* 📋 View all tasks

---

## 🧠 Concepts Used

* Python Lists
* While Loop
* If-Else Conditions
* Functions

---

## 🔄 Workflow Explanation

1. User enters number of tasks
2. Tasks are stored in a list
3. Menu is displayed continuously
4. User selects operation:

   * Add → append task
   * Update → replace task
   * Delete → remove task
   * View → display all tasks
5. Loop runs until user exits
