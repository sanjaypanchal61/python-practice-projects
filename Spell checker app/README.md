# Spell Checker App

## Project Overview

This is a command-line based Spell Checker application built using Python.
It takes user input text and automatically corrects spelling mistakes using a spell-checking library.

---

## Objectives

* Learn how to use external Python libraries
* Implement text processing

---

## Concepts Used

* Python Classes & Objects (OOP)
* External Library: `pyspellchecker`
* String manipulation
* Loops & conditions

---

## Installation

```bash id="9mdh9k"
pip install pyspellchecker
```
