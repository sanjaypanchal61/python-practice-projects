# ⏰ Digital Clock

## 📌 Project Overview

This is a simple Digital Clock GUI application built using Python and Tkinter.

---

## 🎯 Objectives

* Learn GUI development using Tkinter
* Understand real-time updates using `.after()`
* Work with time formatting using `strftime()`

---

## ⚙️ Features

* 🕒 Real-time clock (updates every second)
* 📅 Displays current date
* 🎨 Simple and clean UI
* ⚡ Lightweight application

---

## 🧠 Concepts Used

* Tkinter (GUI Library)
* `strftime()` for time formatting
* `.after()` method (loop alternative)
