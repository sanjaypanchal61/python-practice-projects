# 🎓 Student Grade Management System

## 📌 Project Overview

This is a command-line based Student Grade Management System built using Python.
It allows users to manage student records with their grades using basic CRUD operations.

---

## ⚙️ Features

* ➕ Add student with grade
* ✏️ Update student grade
* ❌ Delete student record
* 📋 View all students
* 🔁 Menu-driven interface

---

## 🧠 Concepts Used

* Python Dictionary (Key-Value pairs)
* Functions (Modular programming)
* Loops (`while`)
* Conditional statements

---

## 💻 Sample Output

```
1. Add
2. Update
3. Delete
4. View
5. Exit
```

---

## 🔄 Workflow

1. User selects operation
2. Input is taken (name, grade)
3. Data stored in dictionary
4. Operation performed (CRUD)
5. Output displayed
